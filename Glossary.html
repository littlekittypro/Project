Web Development Glossary

HTML
HyperText Markup Language, used to structure content on the web.
CSS
Cascading Style Sheets, used for designing and styling web pages.
JavaScript
A scripting language used to create dynamic website features.
DOM
Document Object Model, a programming interface for HTML and XML documents.
API
Application Programming Interface, a set of rules for interacting with software applications.
Bootstrap
A front-end framework for developing responsive and mobile-first websites.
Git
A version control system for tracking changes in code during development.
Responsive Design
Design approach ensuring web content adapts to different screen sizes and devices.
